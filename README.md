# BlinkIT Grocery Sales Analysis

## 🚀 Project Overview
This project provides a comprehensive end-to-end analysis of BlinkIT's retail performance, analyzing **$1.20M in total sales revenue**. The objective was to build a data pipeline that transforms raw transactional data into actionable business intelligence through data cleaning, SQL analysis, and interactive dashboarding.

---

## 🛠 Tech Stack
* **Language:** Python (Pandas, NumPy, Matplotlib, Seaborn)
* **Database:** PostgreSQL (SQLAlchemy)
* **Visualization:** Power BI

---

## 📊 Key Insights
* **Total Revenue:** Analyzed a total sales volume of **$1.20M** across 8,523 items.
* **Top Performance:** **Supermarket Type1** in **Tier 2** regions serves as the most efficient channel, generating **$393.15K**.
* **Product Leaders:** **Fruits & Vegetables** and **Snack Foods** are the primary revenue pillars, each contributing **$0.18M**.
* **Market Preference:** Health-conscious consumers are a significant segment, with **Low Fat products** holding a **64.6% market share**.

---

## 💡 Strategic Recommendations
1. **Regional Expansion:** Prioritize infrastructure rollout in **Tier 3** corridors, as they represent the highest gross revenue source ($472.13K).
2. **Category Management:** Coordinate multi-category bundle marketing campaigns between **Fruits & Vegetables** and **Snack Foods** to maximize transaction throughput.
3. **Inventory Strategy:** Maintain persistent safety stock levels for **Low Fat** products to prevent revenue loss from out-of-stock events.
